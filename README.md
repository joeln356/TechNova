### ✨ Sobre o TechNova

**TechNova** é uma landing page limpa e moderna, criada para apresentar **ideias inovadoras**, startups tecnológicas,com uma estética futurista/neon.

Principais características:
- Design responsivo (mobile-first)
- Animações suaves usando apenas CSS
- Efeitos modernos de hover e scroll
- Carregamento rápido (sem frameworks pesados)
